# Titanic Survival Prediction

My first end-to-end Machine Learning project.

**Accuracy**: 82.12% using XGBoost

### What I learned
- Full ML pipeline (EDA → Cleaning → Feature Engineering → Modeling → Evaluation)
- Importance of feature engineering (Title, FamilySize, IsWomanOrChild, RichWoman)
- How to handle missing values and categorical data
- Using XGBoost with GridSearchCV

### Technologies
- Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Seaborn
